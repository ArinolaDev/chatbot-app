# 🛒 AI E-Commerce Chatbot

This is an AI-powered customer support chatbot built using Streamlit and TensorFlow.

## 🚀 Features
- Chat-based interface
- Intent recognition using machine learning
- Product recommendation system
- Simple and interactive UI

## 🧠 Technologies Used
- Python
- Streamlit
- TensorFlow
- NLTK
- NumPy

## 📦 How to Run
1. Install dependencies:
   pip install -r requirements.txt

2. Run the app:
   streamlit run Ai.py

## 💬 Example Use Cases
- Customer support chatbot
- E-commerce assistant
- AI demo project
